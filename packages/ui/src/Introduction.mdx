# @plug-atlas/ui Documentation

**실외 지도 기반 관제 시스템 UI 라이브러리**

## 📦 Overview

`@plug-atlas/ui`는 Atomic Design 패턴을 기반으로 구축된 React 컴포넌트 라이브러리입니다.

### 기술 스택
- **React 19** - 최신 React 기능 활용
- **Tailwind CSS v4** - 유틸리티 기반 스타일링
- **Radix UI** - 접근성 우선 프리미티브
- **TypeScript** - 타입 안정성
- **class-variance-authority** - 변형 관리

- Button, Input, Label, Badge, Select
- Checkbox, Switch, Toggle, Radio Group
- Avatar, Progress, Spinner, Skeleton
- Separator, Aspect Ratio, Input OTP

2-3개의 Atoms 조합
- Alert, Card, Tooltip, Popover
- Tabs, Accordion, Toast, Sonner
- Collapsible, Hover Card, Pagination

### Organisms (유기체) - 9개

- Dialog, Alert Dialog, Command
- Combobox, Navigation Menu, Form
- Data Table, Date Picker, Calendar, Chart

## 🚀 Usage

### Installation

```bash
pnpm add @plug-atlas/ui
```

### Import Components

```tsx
import { Button, Input, Dialog } from '@plug-atlas/ui'
import '@plug-atlas/ui/globals.css'

function App() {
  return (
    <div>
      <Button variant="default">Click me</Button>
      <Input placeholder="Enter text..." />
    </div>
  )
}
```

### Import by Category

```tsx
// Atoms만 import
import { Button, Input, Badge } from '@plug-atlas/ui/atoms'

// Molecules만 import
import { Card, Alert } from '@plug-atlas/ui/molecules'

// Organisms만 import
import { Dialog, Select } from '@plug-atlas/ui/organisms'
```

## ✨ Conventions

### Prop Naming
- `variant` → `appearance` (시각적 스타일)
- `size` → `scale` (크기 변형)
- 네이티브 HTML prop과의 충돌 방지

### Data Attributes
모든 컴포넌트에 `data-slot` 속성 적용
- 스타일링 훅
- 분석 추적
- 테스트 선택자

### Accessibility
- ARIA 속성 자동 적용
- 키보드 네비게이션 지원
- 스크린 리더 최적화

## 📖 Documentation

이 Storybook은 **Auto Docs**를 사용하여 자동으로 문서를 생성합니다.

### 문서 구성
- **Description**: 컴포넌트 설명 (stories의 `parameters.docs.description.component`)
- **Controls**: Props를 실시간으로 조작하며 테스트
- **Stories**: 다양한 사용 예제 (Canvas에서 Preview + Code)
- **Props Table**: 자동 생성된 Props 타입 정보

### 새 컴포넌트 문서 작성
```tsx
const meta: Meta<typeof Component> = {
  title: 'Category/ComponentName',
  component: Component,
  parameters: {
    docs: {
      description: {
        component: '컴포넌트에 대한 상세 설명...',
      },
    },
  },
}
```
